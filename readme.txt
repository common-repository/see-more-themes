See more themes when viewing installed themes or adding new ones.

== Description ==

When viewing installed themes or searching for new themes in your WordPress dashboard, See More Themes allows you to... see more themes... on the screen at once.

Follow See More Themes' development on [Github](https://github.com/sdavismedia/see-more-themes)

== Installation ==

1. Upload `see-more-themes` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the dashboard -> Themes page to see the difference.

== Frequently Asked Questions ==

= Are there plugin settings? =

No. Once activated, it simply works.
